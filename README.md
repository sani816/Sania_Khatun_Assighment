# 💸 Smart Expense Splitter

A simple and intuitive web application to manage and split shared expenses among friends, roommates, or teams.
This project helps users easily track who paid, who owes, and how to settle balances — all in real time.

## 📌 Features

### ✅ Core Features

* 👥 Create group and add members
* 💸 Add shared expenses
* ⚖️ Split expenses equally or custom
* 📊 Automatic balance calculation
* 🤝 Settlement system (who pays whom)

### ⭐ Advanced Features

* ⚡ Real-time updates
* 🎨 Modern Neon Glass UI (responsive design)
* 📱 Fully responsive (mobile, tablet, desktop)
* 👆 Clickable payer selection (no dropdown)
* 🧠 Clear UX with instructions

---

## 🛠️ Tech Stack

* **HTML** – Structure
* **CSS** – Styling (Glassmorphism + Responsive Design)
* **JavaScript** – Logic & Functionality

---

## 🧠 How It Works

1. Add members to the group
2. Enter expense details
3. Select who paid
4. Choose equal or custom split
5. App calculates balances instantly
6. View final settlement (who owes whom)
